## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm

### Installation

1. Install server dependencies:
```bash
cd server
npm install
```

2. Install client dependencies:
```bash
cd client
npm install
```

### Running the App

1. Start the backend server:
```bash
cd server
npm run dev
```

2. In a new terminal, start the React app:
```bash
cd client
npm start
```

The app will open at `http://localhost:3000`
